## Summary
This project analyzes how income inequality and national development conditions shape happiness across countries. It combines data from the World Happiness Report, Gini Index, and key development indicators such as GDP per capita, education, health, inflation, and urbanization. The analysis includes data cleaning, exploratory analysis, group comparisons, and multiple regression models with interaction effects. Results show that inequality is weakly but consistently linked to lower happiness, while development indicators explain most of the variation in well‑being. Interaction models reveal that GDP and urbanization significantly moderate the inequality–happiness relationship. All code, datasets, and visualizations used in the analysis are included in this repository.
